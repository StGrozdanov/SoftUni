This exam includes a single problem:

You are given an MVC project skeleton with pre made front end and web controllers, your job is:
1) Create Database Models
2) Implement DTO's
3) Create services
4) Create repositories
5) Import two json and two xml data files into the database with the help of model mapper, gson, jaxb and validation utils
6) Create complex query that exports a given information from the database and visualises it into the front end.
